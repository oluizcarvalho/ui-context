# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2025-03-29

### Added

- Core schema and parser interface (`@ui-context/core`)
- React component parser using react-docgen (`@ui-context/parser-react`)
- Angular component parser using ts-morph (`@ui-context/parser-angular`)
- MCP Server runtime with 5 tools (`@ui-context/mcp-server`)
- CLI tool with generate, serve, and validate commands (`@ui-context/cli`)
- Documentation site with dark theme (GitHub Pages)
- Example React and Angular component libraries
